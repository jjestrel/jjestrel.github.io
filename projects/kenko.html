What is Kenko?

Kenko is an intelligent meal planner that takes the hassle out of planning nutritionally complete and delicious meals. More information can be found on the website.

What Runs Kenko?

On the backend we're using Laravel 5, MySQL and Mandrill. On the frontend right now we're using bootstrap.
